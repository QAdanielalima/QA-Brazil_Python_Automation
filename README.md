Projeto de automação de testes end-to-end desenvolvido em Python utilizando Selenium WebDriver, com foco na validação das principais funcionalidades de um aplicativo de corridas urbanas (Urban Routes).

A automação foi estruturada seguindo o padrão Page Object Model (POM), garantindo organização, reutilização de código e facilidade de manutenção. Foram implementados testes para fluxos críticos como solicitação de corrida, seleção de rotas e interação com elementos dinâmicos da interface.

Além disso, foram aplicadas boas práticas de automação, como uso de waits explícitos para lidar com sincronização, separação de responsabilidades e criação de localizadores robustos.

O projeto tem como objetivo garantir a estabilidade das funcionalidades principais da aplicação e simular o comportamento real do usuário final.
